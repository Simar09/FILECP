# FileCP Support

Welcome to the FileCP Support page.

If you experience any issues, have questions, would like to report a bug, or have suggestions for improving FileCP, please don't hesitate to get in touch.

## Support Topics

You can contact us regarding:

- Installation and setup issues
- File transfer problems
- QR code connection issues
- Bug reports
- Feature requests
- Performance issues
- Security concerns
- General feedback

## Before Contacting Support

To help us resolve your issue quickly, please include:

- Your operating system (Windows version)
- Browser name and version (Chrome or Microsoft Edge)
- FileCP version
- A detailed description of the issue
- Steps to reproduce the problem
- Screenshots or error messages (if available)

## Response Time

We aim to respond to all support requests within **24–72 hours**.

## Feedback

Your feedback helps improve FileCP. We appreciate your suggestions and ideas for future updates.

Thank you for using **FileCP**!
